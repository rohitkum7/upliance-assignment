# Recipe App

A fully functional **Recipe Management Application** built using **React**, **Redux Toolkit**, and **Material UI (MUI)**.  
This project allows users to create, view, filter, sort, and favorite recipes with a delightful and modern UI.

## Project Overview

This app simulates a modern recipe manager, where users can:

- Add and edit recipes
- Filter by difficulty and sort by total cook time
- Mark recipes as favorites
- Start a cooking session and follow step-by-step instructions
